Refactor this Python code to be more idiomatic:
```python
def find_max(numbers):
    max_val = numbers[0]
    for i in range(1, len(numbers)):
        if numbers[i] > max_val:
            max_val = numbers[i]
    return max_val

result = find_max([3, 7, 2, 9, 1, 5])
```
