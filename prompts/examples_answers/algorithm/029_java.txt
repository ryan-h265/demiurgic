```java
import java.util.*;

public class TrappingRainWater {
    // Two-pointer approach to compute trapped water
    public static int trap(int[] height) {
        int left = 0, right = height.length - 1;
        int leftMax = 0, rightMax = 0, water = 0;

        while (left < right) {
            if (height[left] < height[right]) {
                if (height[left] >= leftMax) {
                    leftMax = height[left];
                } else {
                    water += leftMax - height[left];
                }
                left++;
            } else {
                if (height[right] >= rightMax) {
                    rightMax = height[right];
                } else {
                    water += rightMax - height[right];
                }
                right--;
            }
        }
        return water;
    }
}
```
**Explanation:** Use two pointers moving inward from both ends. Track the highest wall seen from each side (`leftMax`, `rightMax`). At each step, move the pointer with the smaller height because its max bound is known; accumulate water as the difference between the current bound and the bar height.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6
**Notes:**
- Handles empty or single-element arrays by returning 0.
- Avoids overflow by using `int`; use `long` if heights may sum beyond `Integer.MAX_VALUE`.
- Assumes non-negative heights.
