```java
public class MedianOfTwoSortedArrays {
    // Binary search partition approach
    public static double findMedianSortedArrays(int[] nums1, int[] nums2) {
        if (nums1.length > nums2.length) {
            return findMedianSortedArrays(nums2, nums1); // ensure nums1 is smaller
        }

        int m = nums1.length, n = nums2.length;
        int low = 0, high = m;

        while (low <= high) {
            int partitionA = (low + high) / 2;
            int partitionB = (m + n + 1) / 2 - partitionA;

            int maxLeftA = partitionA == 0 ? Integer.MIN_VALUE : nums1[partitionA - 1];
            int minRightA = partitionA == m ? Integer.MAX_VALUE : nums1[partitionA];
            int maxLeftB = partitionB == 0 ? Integer.MIN_VALUE : nums2[partitionB - 1];
            int minRightB = partitionB == n ? Integer.MAX_VALUE : nums2[partitionB];

            if (maxLeftA <= minRightB && maxLeftB <= minRightA) {
                if ((m + n) % 2 == 0) {
                    return (Math.max(maxLeftA, maxLeftB) + Math.min(minRightA, minRightB)) / 2.0;
                }
                return Math.max(maxLeftA, maxLeftB);
            }

            if (maxLeftA > minRightB) {
                high = partitionA - 1; // move left
            } else {
                low = partitionA + 1; // move right
            }
        }

        throw new IllegalArgumentException("Arrays must be sorted");
    }
}
```
**Explanation:** Binary-search the smaller array for a partition that splits the combined arrays into equal (or nearly equal) halves where all left elements are â‰¤ right elements. Use sentinel values at boundaries. When the correct partition is found, compute the median from border elements depending on parity of total length.
**Complexity:** Time: O(log(min(m, n))); Space: O(1).
**Example:**
Input: nums1 = [1, 3], nums2 = [2, 4]
Output: 2.5
**Notes:**
- Handles empty arrays due to sentinel values.
- Throws when inputs are not sorted or sizes zero with k invalid.
- Uses integer sentinels; adjust to long if values may hit Integer limits.
