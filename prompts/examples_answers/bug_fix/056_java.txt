```java
public class SafePrinter {
    // Fix: iterate with i < arr.length to avoid out-of-bounds, handle empty arrays
    public static void print(int[] arr) {
        if (arr == null || arr.length == 0) {
            return;
        }
        for (int i = 0; i < arr.length; i++) {
            System.out.println(arr[i]);
        }
    }
}
```
**Explanation:** Corrects the loop condition from `<= length` to `< length`, preventing access past the last element. It also guards against null or empty arrays so n = 0 is safe.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: arr = {10, 20}
Output: prints 10 then 20
**Notes:**
- Replace `int` with the desired type as needed.
- Early return keeps behavior no-op for missing data rather than throwing exceptions.
- For logging, consider using a logger instead of `System.out`.
