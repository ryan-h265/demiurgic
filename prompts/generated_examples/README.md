# Example Coding Prompts

This directory contains 500 example coding prompts generated by Demiurgic.

## Files

- `all_prompts.json` - All prompts in JSON format
- `all_prompts.jsonl` - All prompts in JSONL format (one per line)
- `<category>/` - Individual prompt files organized by category

## Categories

1. **function_implementation** - Basic function writing tasks
2. **algorithm** - Algorithm implementation (sorting, searching, etc.)
3. **data_structure** - Data structure implementation
4. **explanation** - Code explanation and understanding
5. **bug_fix** - Find and fix bugs in code
6. **refactoring** - Improve code quality
7. **real_world** - Practical tasks (file I/O, APIs, etc.)
8. **code_review** - Code review and optimization

## Usage

Use these prompts to:
- Test your model's code generation capabilities
- Understand the types of tasks in training data
- Generate responses from teacher models
- Create evaluation benchmarks

## Example

```python
import json

# Load all prompts
with open('all_prompts.json', 'r') as f:
    prompts = json.load(f)

# Use a prompt
prompt = prompts[0]['prompt']
print(prompt)
```
