```java
public class SafeArrayPrinter {
    public static void print(int[] arr) {
        if (arr == null) return;
        for (int i = 0; i < arr.length; i++) {
            System.out.println(arr[i]);
        }
    }
}
```
**Explanation:** Fixes the off-by-one loop (`<= arr.length`) that accessed beyond the array. Uses `< arr.length` and guards null input.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: [1, 2, 3]
Output: prints 1, 2, 3 on separate lines
**Notes:**
- For empty arrays, the loop runs zero times safely.
- Replace `System.out` with a logger for production code.
- Add bounds checks if user-provided indices are involved.
