```java
public class SafeCycle {
    static class ListNode { int val; ListNode next; }

    public static boolean hasCycle(ListNode head) {
        ListNode slow = head, fast = head;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
            if (slow == fast) return true;
        }
        return false;
    }
}
```
**Explanation:** Adds null checks before advancing `fast.next.next` to avoid null-pointer errors when the list is empty (n=0) or short. Uses Floyd's algorithm to detect cycles safely.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: 1 -> 2 -> 3 -> 2
Output: true
**Notes:**
- Works for empty and single-node lists (returns false).
- To find cycle entry, reset one pointer to head after detection and move both one step at a time.
- Leaves list unchanged.
