```javascript
function indexOfFixed(nums, target) {
  if (!Array.isArray(nums)) return -1;
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] === target) return i;
  }
  return -1;
}
```
**Explanation:** Ensures the loop scans all elements using `i < nums.length` and only returns -1 after the search is exhausted, fixing the premature return in the original snippet.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: nums = [5, 6], target = 7
Output: -1
**Notes:**
- Returns the first occurrence; adjust if last is needed.
- Works for empty arrays safely.
- Strict equality avoids coercion surprises.
