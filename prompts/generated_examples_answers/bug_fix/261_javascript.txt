```javascript
function indexOfSafe(nums, target) {
  if (!Array.isArray(nums)) return -1;
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] === target) return i;
  }
  return -1;
}
```
**Explanation:** Checks that input is an array, iterates with `i < nums.length`, and returns -1 only after scanning all elements. This avoids null/undefined dereferences and out-of-bounds access.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: nums = [1, 2, 3], target = 4
Output: -1
**Notes:**
- For repeated lookups, use a map to speed up searches.
- Strict equality prevents type-coercion surprises; adjust if needed.
- Empty arrays return -1 immediately.
