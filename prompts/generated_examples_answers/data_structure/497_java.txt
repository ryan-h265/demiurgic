```java
public class UnionFindLRU {
    private final int[] parent;
    private final int[] rank;

    public UnionFindLRU(int n) {
        parent = new int[n];
        rank = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
    }

    public int find(int x) {
        if (parent[x] != x) parent[x] = find(parent[x]);
        return parent[x];
    }

    public void union(int a, int b) {
        int ra = find(a), rb = find(b);
        if (ra == rb) return;
        if (rank[ra] < rank[rb]) { int t = ra; ra = rb; rb = t; }
        parent[rb] = ra;
        if (rank[ra] == rank[rb]) rank[ra]++;
    }
}
```
**Explanation:** Union-find with path compression and union by rank for near-constant-time connectivity operations. LRU mention is not relevant; this provides disjoint-set functionality.
**Complexity:** Time: Amortized ~O(Î±(n)); Space: O(n).
**Example:**
Input: union(0,1); union(2,3); find(3)
Output: representative of set containing 3
