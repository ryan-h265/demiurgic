```javascript
class MinHeap {
  constructor() {
    this.data = [];
  }
  push(x) {
    this.data.push(x);
    this._siftUp(this.data.length - 1);
  }
  pop() {
    if (this.data.length === 0) return undefined;
    const root = this.data[0];
    const last = this.data.pop();
    if (this.data.length && last !== undefined) {
      this.data[0] = last;
      this._siftDown(0);
    }
    return root;
  }
  peek() {
    return this.data[0];
  }
  _siftUp(i) {
    while (i > 0) {
      const p = (i - 1) >> 1;
      if (this.data[p] <= this.data[i]) break;
      [this.data[p], this.data[i]] = [this.data[i], this.data[p]];
      i = p;
    }
  }
  _siftDown(i) {
    const n = this.data.length;
    while (true) {
      let smallest = i;
      const l = 2 * i + 1, r = 2 * i + 2;
      if (l < n && this.data[l] < this.data[smallest]) smallest = l;
      if (r < n && this.data[r] < this.data[smallest]) smallest = r;
      if (smallest === i) break;
      [this.data[i], this.data[smallest]] = [this.data[smallest], this.data[i]];
      i = smallest;
    }
  }
}
```
**Explanation:** Implements a binary min-heap with array storage. `push` appends then sifts up; `pop` removes root, replaces with last, then sifts down to restore heap order. `peek` inspects the minimum without removal.
**Complexity:** Time: O(log n) for push/pop; O(1) for peek; Space: O(n).
**Example:**
Input: push 3, push 1, push 2, pop()
Output: pop returns 1
**Notes:**
- Works for numbers; supply a comparator if you need custom ordering.
- No capacity limit; add one if needed to bound memory.
- Ensure stability if storing objects by wrapping with priority values.
