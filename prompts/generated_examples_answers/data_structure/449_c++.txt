```cpp
#include <map>
#include <string>

class Trie {
    struct Node {
        std::map<char, Node*> children;
        bool end = false;
    };
    Node* root;
public:
    Trie() : root(new Node()) {}
    void insert(const std::string& word) {
        Node* cur = root;
        for (char ch : word) {
            if (!cur->children.count(ch)) cur->children[ch] = new Node();
            cur = cur->children[ch];
        }
        cur->end = true;
    }
    bool search(const std::string& word) const {
        const Node* cur = root;
        for (char ch : word) {
            auto it = cur->children.find(ch);
            if (it == cur->children.end()) return false;
            cur = it->second;
        }
        return cur->end;
    }
};
```
**Explanation:** Trie with ordered children via `std::map`, supporting insertion and exact-word search. Nodes are dynamically allocated and marked when a word ends.
**Complexity:** Time: O(L) per insert/search; Space: O(total characters stored).
**Example:**
Input: insert("cat"); search("cat")
Output: true
**Notes:**
- Add destructor to free nodes in production.
- Replace `std::map` with `std::unordered_map` or array for performance.
- Add prefix search, delete, or min/max retrieval as needed.
