```python
import threading
from collections import OrderedDict

class ThreadSafeLRU:
    def __init__(self, capacity: int):
        if capacity <= 0:
            raise ValueError("capacity must be positive")
        self.cap = capacity
        self.cache: OrderedDict[int, int] = OrderedDict()
        self.lock = threading.Lock()

    def get(self, key):
        with self.lock:
            if key not in self.cache:
                return -1
            self.cache.move_to_end(key)
            return self.cache[key]

    def put(self, key, value):
        with self.lock:
            if key in self.cache:
                self.cache.move_to_end(key)
            self.cache[key] = value
            if len(self.cache) > self.cap:
                self.cache.popitem(last=False)
```
**Explanation:** Wraps `OrderedDict` operations with a mutex. `get` updates recency, `put` inserts/updates and evicts the least recently used when exceeding capacity.
**Complexity:** Time: O(1) average per operation; Space: O(capacity).
**Example:**
Input: cap=2; put(1,1); put(2,2); get(1); put(3,3); get(2)
Output: get(1)=1, get(2)=-1 (evicted)
**Notes:**
- Returns -1 on misses; adjust to raise if preferred.
- For high contention, consider finer-grained or lock-free structures, but OrderedDict access is simple and reliable.
- Ensure keys are hashable; change types as needed.
