```python
class UnionFind:
    def __init__(self, n: int):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, x: int) -> int:
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]

    def union(self, a: int, b: int) -> None:
        ra, rb = self.find(a), self.find(b)
        if ra == rb:
            return
        if self.rank[ra] < self.rank[rb]:
            ra, rb = rb, ra
        self.parent[rb] = ra
        if self.rank[ra] == self.rank[rb]:
            self.rank[ra] += 1
```
**Explanation:** Disjoint set union with path compression and union by rank for near-constant-time connectivity operations.
**Complexity:** Time: Amortized ~O(Î±(n)) per op; Space: O(n).
**Example:**
Input: uf.union(0,1); uf.union(1,2); uf.find(2)
Output: representative of {0,1,2}
**Notes:**
- No range queries inherently; to support extra metadata per set, store additional arrays keyed by roots.
- Validate indices before use in production.
- Add a `connected(a,b)` helper for convenience.
