```java
import java.util.*;

public class WordLadder {
    public static int ladderLength(String beginWord, String endWord, List<String> wordList) {
        Set<String> dict = new HashSet<>(wordList);
        if (!dict.contains(endWord)) return 0;
        Queue<String> q = new ArrayDeque<>();
        q.add(beginWord);
        Set<String> visited = new HashSet<>();
        visited.add(beginWord);
        int steps = 1;
        while (!q.isEmpty()) {
            int sz = q.size();
            for (int i = 0; i < sz; i++) {
                String word = q.poll();
                if (word.equals(endWord)) return steps;
                char[] chars = word.toCharArray();
                for (int pos = 0; pos < chars.length; pos++) {
                    char original = chars[pos];
                    for (char c = 'a'; c <= 'z'; c++) {
                        if (c == original) continue;
                        chars[pos] = c;
                        String next = new String(chars);
                        if (dict.contains(next) && visited.add(next)) {
                            q.add(next);
                        }
                    }
                    chars[pos] = original;
                }
            }
            steps++;
        }
        return 0;
    }
}
```
**Explanation:** BFS over one-letter transformations contained in the dictionary. Tracks visited words to avoid cycles; the first time `endWord` is reached yields the shortest ladder length.
**Complexity:** Time: O(N * L * 26) where N is dictionary size and L word length; Space: O(N) for visited/queue.
**Example:**
Input: begin="hit", end="cog", list=["hot","dot","dog","lot","log","cog"]
Output: 5
**Notes:**
- Returns 0 if no transformation exists or endWord missing.
- For speed on large dicts, use bidirectional BFS.
- Assumes all words are same length; filter otherwise.
