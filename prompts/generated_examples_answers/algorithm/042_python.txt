```python
from typing import List

def rotate(matrix: List[List[int]]) -> None:
    """Rotate the square matrix 90 degrees clockwise in-place."""
    n = len(matrix)
    # transpose
    for i in range(n):
        for j in range(i + 1, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
    # reverse each row
    for row in matrix:
        row.reverse()
```
**Explanation:** Transpose swaps rows and columns, then reversing each row yields a 90° clockwise rotation. Both steps are in-place, requiring only constant extra space.
**Complexity:** Time: O(n²); Space: O(1) extra.
**Example:**
Input: [[1,2],[3,4]]
Output: [[3,1],[4,2]]
**Notes:**
- Matrix must be square; pad or adjust for rectangular shapes.
- For counterclockwise rotation, transpose then reverse columns instead.
- Validate non-empty input to avoid index errors.
