```java
public class ValidateBST {
    static class TreeNode { int val; TreeNode left, right; TreeNode(int v){val=v;} }

    public static boolean isValidBST(TreeNode root) {
        return helper(root, null, null);
    }
    private static boolean helper(TreeNode node, Integer low, Integer high) {
        if (node == null) return true;
        if ((low != null && node.val <= low) || (high != null && node.val >= high)) return false;
        return helper(node.left, low, node.val) && helper(node.right, node.val, high);
    }
}
```
**Explanation:** Recursively enforces BST bounds: each node must be greater than all left ancestors (low) and less than all right ancestors (high). Any violation returns false.
**Complexity:** Time: O(n); Space: O(h) recursion stack.
**Example:**
Input: [2,1,3]
Output: true
**Notes:**
- Strict inequalities forbid duplicates; adjust if duplicates allowed on one side.
- Use iterative inorder traversal if concerned about stack depth.
- Ensure TreeNode structure matches your environment.
