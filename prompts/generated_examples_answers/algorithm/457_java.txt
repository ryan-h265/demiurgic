```java
import java.util.*;

public class LongestIncreasingSubsequence {
    public static int lengthOfLIS(int[] nums) {
        int[] tails = new int[nums.length];
        int size = 0;
        for (int x : nums) {
            int i = Arrays.binarySearch(tails, 0, size, x);
            if (i < 0) i = -i - 1; // lower bound
            tails[i] = x;
            if (i == size) size++;
        }
        return size;
    }
}
```
**Explanation:** Maintains `tails[len]` as the smallest tail of an increasing subsequence of length `len+1`. For each number, binary search finds its position; replacing tails preserves minimal tails, and extending increases LIS length.
**Complexity:** Time: O(n log n); Space: O(n) for tails.
**Example:**
Input: [10,9,2,5,3,7,101,18]
Output: 4
**Notes:**
- For non-decreasing LIS, adjust binary search to upper_bound.
- To reconstruct the sequence, track predecessors alongside tails indices.
- Handles empty input (size remains 0).
