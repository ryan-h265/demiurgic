```java
public class PalindromeChecker {
    public static boolean isPalindrome(String s) {
        if (s == null) return false;
        int i = 0, j = s.length() - 1;
        while (i < j) {
            if (s.charAt(i) != s.charAt(j)) return false;
            i++; j--;
        }
        return true;
    }
}
```
**Explanation:** Two-pointer comparison from both ends of the string; returns false on first mismatch, true if all pairs match.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: "level"
Output: true
**Notes:**
- Case-sensitive; add normalization if needed.
- For ignoring non-alphanumerics, skip them within the loop.
- Returns false on null to avoid ambiguity; adjust to throw if preferred.
