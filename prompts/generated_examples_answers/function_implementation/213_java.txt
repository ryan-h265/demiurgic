```java
import java.util.regex.Pattern;

public class EmailValidator {
    private static final Pattern EMAIL = Pattern.compile("^[\\w.+-]+@[A-Za-z0-9.-]+\\.[A-Za-z]{2,}$");

    public static boolean isValid(String email) {
        return email != null && EMAIL.matcher(email.trim()).matches();
    }
}
```
**Explanation:** Precompiles a simple regex for common email shapes (local@domain.tld). `isValid` trims whitespace, checks null, and matches against the pattern.
**Complexity:** Time: O(n) for match; Space: O(1) auxiliary.
**Example:**
Input: "user@example.com"
Output: true
**Notes:**
- Regex is intentionally permissive; full RFC compliance requires a more complex pattern.
- Trim handling avoids leading/trailing whitespace issues.
- Consider normalizing case for domain comparisons if needed.
