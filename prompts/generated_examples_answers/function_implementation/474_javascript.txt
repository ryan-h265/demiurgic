```javascript
// Retry an async function with exponential backoff
async function retryWithBackoff(fn, retries = 3, baseDelayMs = 200) {
  for (let attempt = 0; attempt <= retries; attempt++) {
    try {
      return await fn();
    } catch (err) {
      if (attempt === retries) throw err;
      const delay = baseDelayMs * Math.pow(2, attempt);
      await new Promise(res => setTimeout(res, delay));
    }
  }
}
```
**Explanation:** Wraps an async operation in a retry loop, doubling the delay after each failure. Returns on the first success or rethrows after exhausting attempts.
**Complexity:** Time: O(retries * fn_time); Space: O(1).
**Example:**
Input: `retryWithBackoff(() => fetch(url).then(r => r.json()))`
Output: JSON response or thrown error after retries
**Notes:**
- Add jitter to delays to avoid synchronized retries under load.
- Filter retryable errors/status codes within `fn` or around this wrapper.
- For idempotent operations; avoid retrying non-idempotent side effects.
