```cpp
#include <string>
#include <cctype>

bool isPalindrome(const std::string& s) {
    int i = 0, j = static_cast<int>(s.size()) - 1;
    while (i < j) {
        while (i < j && !std::isalnum(static_cast<unsigned char>(s[i]))) i++;
        while (i < j && !std::isalnum(static_cast<unsigned char>(s[j]))) j--;
        if (std::tolower(static_cast<unsigned char>(s[i])) != std::tolower(static_cast<unsigned char>(s[j]))) {
            return false;
        }
        i++; j--;
    }
    return true;
}
```
**Explanation:** Two-pointer scan from both ends, skipping non-alphanumeric characters and comparing case-insensitively. Returns false on the first mismatch.
**Complexity:** Time: O(n); Space: O(1).
**Example:**
Input: "A man, a plan, a canal: Panama"
Output: true
**Notes:**
- Remove skipping/normalization if punctuation/case should be considered.
- Works for empty or single-character strings (palindromes by definition).
- Cast to unsigned char before `std::tolower`/`std::isalnum` to avoid UB.
