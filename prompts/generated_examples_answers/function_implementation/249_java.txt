```java
public class GCDUtil {
    public static int gcd(int a, int b) {
        a = Math.abs(a);
        b = Math.abs(b);
        while (b != 0) {
            int tmp = a % b;
            a = b;
            b = tmp;
        }
        return a;
    }
}
```
**Explanation:** Uses the iterative Euclidean algorithm, swapping remainder into the loop until the second operand becomes zero. Absolute values handle negative inputs.
**Complexity:** Time: O(log min(a,b)); Space: O(1).
**Example:**
Input: gcd(48, 18)
Output: 6
**Notes:**
- If both inputs are zero, returns zero; adjust behavior if needed.
- For multiple numbers, fold gcd across the list.
- Use `long` for larger ranges to avoid overflow.
