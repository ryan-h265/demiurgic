```java
public class QuickSort {
    public static void quicksort(int[] arr) {
        sort(arr, 0, arr.length - 1);
    }
    private static void sort(int[] arr, int lo, int hi) {
        if (lo >= hi) return;
        int p = partition(arr, lo, hi);
        sort(arr, lo, p - 1);
        sort(arr, p + 1, hi);
    }
    private static int partition(int[] arr, int lo, int hi) {
        int pivot = arr[hi];
        int i = lo;
        for (int j = lo; j < hi; j++) {
            if (arr[j] <= pivot) {
                swap(arr, i, j);
                i++;
            }
        }
        swap(arr, i, hi);
        return i;
    }
    private static void swap(int[] arr, int i, int j) {
        int tmp = arr[i]; arr[i] = arr[j]; arr[j] = tmp;
    }
}
```
**Explanation:** Lomuto partition quicksort: choose the last element as pivot, partition the array around it, then recursively sort subarrays left/right of the pivot index.
**Complexity:** Time: O(n log n) average, O(nÂ²) worst case; Space: O(log n) recursion stack.
**Example:**
Input: [3, 7, 2, 5]
Output: [2, 3, 5, 7]
**Notes:**
- Randomize pivot to reduce worst-case likelihood on sorted input.
- For tiny partitions, switching to insertion sort can improve constants.
- Not stable; equal elements may reorder.
