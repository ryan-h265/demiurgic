```java
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
@RequestMapping("/api")
public class HelloController {

    @GetMapping("/hello")
    public Message hello() {
        return new Message("Hello, world");
    }

    public record Message(String message) {}
}
```
**Explanation:** Defines a Spring REST controller with a `/api/hello` GET endpoint that returns a JSON message. Uses a record for the response payload.
**Complexity:** Time: O(1); Space: O(1) beyond framework overhead.
**Example:**
Input: GET /api/hello
Output: {"message":"Hello, world"}
**Notes:**
- Requires Spring Web dependency and appropriate application bootstrap.
- Add validation/authentication as needed for real endpoints.
- Change mappings to fit your routing conventions.
