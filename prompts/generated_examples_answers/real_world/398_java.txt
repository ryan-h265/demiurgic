```java
import java.util.Arrays;

public class CliArgs {
    public static void main(String[] args) {
        if (args.length == 0) {
            System.out.println("Usage: --name <value> [--verbose]");
            return;
        }
        String name = null; boolean verbose = false;
        for (int i = 0; i < args.length; i++) {
            switch (args[i]) {
                case "--name" -> {
                    if (i + 1 >= args.length) throw new IllegalArgumentException("--name requires a value");
                    name = args[++i];
                }
                case "--verbose" -> verbose = true;
                default -> throw new IllegalArgumentException("Unknown flag: " + args[i]);
            }
        }
        System.out.printf("name=%s verbose=%b%n", name, verbose);
    }
}
```
**Explanation:** Minimal CLI parser: scans argv for `--name` (expects a following value) and `--verbose` flag, handling errors for unknown flags or missing values, then prints parsed results.
**Complexity:** Time: O(n) over arguments; Space: O(1).
**Example:**
Input: `java CliArgs --name Alice --verbose`
Output: `name=Alice verbose=true`
**Notes:**
- For richer CLIs, use a library like picocli.
- Add default values and required-field checks as appropriate.
- Expand switch with more flags or subcommands as needed.
